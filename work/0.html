Warm Up Bot

Think less, do more

I built a tool I found really useful for my own creative progression.

It’s called WarmUpBot: with Github actions and a CSV file, you can generate a personalised list of warm-up exercises to execute every day.

I built this because as an artist, I sometimes sit down to get something done and struggle to get started. I find doing warm-ups really helps me out and this meets my own need by giving me different exercises to warm up against.

It works by selecting random tasks from a .csv file and every day, you’ll get a new list of To-Dos to get you started.

I hope you enjoy it and I am excited to see what exercises people use to warm up!

The link can be found here: https://github.com/vikadilly/warmupbot
